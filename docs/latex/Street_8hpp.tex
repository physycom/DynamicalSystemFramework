\doxysubsection{Detailed Description}
Defines the Street class. 

This file contains the definition of the Street class. The Street class represents a street in the network. It is templated by the type of the street\textquotesingle{}s id and the type of the street\textquotesingle{}s capacity. The street\textquotesingle{}s id and capacity must be unsigned integral types.
