Defines the Sparse\+Matrix class. 

This file contains the definition of the Sparse\+Matrix class. The Sparse\+Matrix class represents a sparse matrix. It is templated by the type of the matrix\textquotesingle{}s index and the type of the matrix\textquotesingle{}s value. The matrix\textquotesingle{}s index must be an unsigned integral type.
