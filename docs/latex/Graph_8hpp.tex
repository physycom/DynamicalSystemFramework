\doxysubsection{Detailed Description}
Defines the Graph class. 

This file contains the definition of the Graph class. The Graph class represents a graph in the network. It is templated by the type of the graph\textquotesingle{}s id and the type of the graph\textquotesingle{}s capacity. The graph\textquotesingle{}s id and capacity must be unsigned integral types.
