\doxysubsection{Detailed Description}
Defines the Itinerary class. 

This file contains the definition of the Itinerary class. The Itinerary class represents an itinerary in the network. It is templated by the type of the itinerary\textquotesingle{}s id. The itinerary\textquotesingle{}s id must be an unsigned integral type.
