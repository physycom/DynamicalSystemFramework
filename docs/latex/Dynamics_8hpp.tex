\doxysubsection{Detailed Description}
Defines the Dynamics class. 

This file contains the definition of the Dynamics class. The Dynamics class represents the dynamics of the network. It is templated by the type of the graph\textquotesingle{}s id and the type of the graph\textquotesingle{}s capacity. The graph\textquotesingle{}s id and capacity must be unsigned integral types.
