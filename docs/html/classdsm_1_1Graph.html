Detailed Description

template<typename Id, typename Size>
class dsm::Graph< Id, Size >

The Graph class represents a graph in the network.

Template Parameters
IdThe type of the graph's id. It must be an unsigned integral type.
SizeThe type of the graph's capacity. It must be an unsigned integral type.
