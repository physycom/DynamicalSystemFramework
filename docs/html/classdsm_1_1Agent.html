Detailed Description

template<typename Id, typename Size, typename Delay>
class dsm::Agent< Id, Size, Delay >

The Agent class represents an agent in the network.

Template Parameters
IdThe type of the agent's id. It must be an unsigned integral type.
SizeThe type of the size of a street. It must be an unsigned integral type.
DelayThe type of the agent's delay. It must be a numeric type (see utility/TypeTraits/is_numeric.hpp).
